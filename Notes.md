## **Visual Design Elements:**

### **Color Scheme:**
- **Primary Color:** `#007bff` (Blue) - for active tabs, primary buttons
- **Secondary Color:** `#6c757d` (Gray) - for secondary buttons
- **Danger Color:** `#dc3545` (Red) - for delete buttons
- **Success Color:** `#28a745` (Green) - for success messages
- **Background:** `#f5f5f5` (Light gray)
- **Cards:** `#ffffff` (White)

### **Typography:**
- **Headers:** Segoe UI, 2.5rem (desktop), responsive for mobile
- **Body Text:** Segoe UI, 1rem, line-height: 1.6
- **Form Labels:** 500 weight, `#495057`
- **Table Text:** 14px

### **UI Components:**
1. **Tab Navigation:** Horizontal tabs with active state indicator
2. **Cards:** Rounded corners with subtle shadows
3. **Forms:** Clean inputs with focus states and validation
4. **Tables:** Striped rows with hover effects
5. **Buttons:** Rounded buttons with hover animations
6. **Modals:** Centered with overlay backdrop
7. **Status Messages:** Color-coded alerts with auto-dismiss
8. **Spinners:** Loading indicators for async operations

### **Interactive Elements:**
- Hover effects on buttons and table rows
- Smooth scrolling when editing
- Form validation with visual feedback
- Confirmation dialogs for destructive actions
- Real-time search results display
- Responsive grid layouts for forms

### **Data Display Patterns:**
1. **Clients:** ID, Name, Phone, City, Country, Actions
2. **Airlines:** ID, Company Name, Actions
3. **Flights:** ID, Client, Airline, Date, Route, Actions
4. **Search Results:** Card-based display with record type badges

This mockup represents a clean, modern, and fully functional interface that matches the Vue.js single-file implementation you requested.
